02 — PROJECTS
⚽
Team Project · 2025

FIFA Players Performance Analysis & Scouting System

End-to-end machine learning pipeline on the FIFA dataset. Predicts player market value (regression) and classifies performance tier — Low / Mid / High / Elite (classification) — using a unified scouting pipeline.

Trained and compared 6+ models: Polynomial Regression (Lasso/Ridge), Logistic Regression, Naive Bayes, KNN, Random Forest, Gradient Boosting
Applied Voting and Stacking ensembles achieving best-in-class accuracy
Full preprocessing pipeline: EDA, IQR outlier removal, StandardScaler, OneHotEncoding, GridSearchCV with 5-fold cross-validation
Unified Scouting Function outputs both market value prediction and performance tier for any new player
Python Scikit-learn Pandas NumPy Matplotlib Seaborn Regression Classification Ensemble Methods
